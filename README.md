# MoviesApp
MovieApp: 
Esta aplicación muestra las peliculas más votadas, los proximos lanzamientos y las más populares.
Utilizando: Kotlin, arquitectura MVVM, API MOVIES DB, recyclerView, Retrofit, navigation Components.
